How It Works:

  • 12-bit Encoding: Tracking ID (0-4095) is split into 4 groups of 3 bits
  • Edge Cropping: Each 3-bit value (0-7) determines pixels removed from each edge
  • Bit Layout: Top(bits 11-9), Right(bits 8-6), Bottom(bits 5-3), Left(bits 2-0)
  • Center Dot: A red dot with white ring marks the exact center for decoding
  • Base Unit: Each bit unit = 2 pixels (max 14px per edge)
  • Compression Resistant: Works even after JPEG compression

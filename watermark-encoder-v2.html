Marker-Only Forensic Watermarking (v2):

  • Single-layer: Only semi-transparent binary grid markers are used — no edge cropping, no center dot.
  • 16-bit capacity: Encodes values 0–65535 using a 5×5 grid (center 3×3 finder pattern + 16 outer data bits).
  • Multiple markers: Several identical markers can appear simultaneously at different positions for greater robustness.
  • Hold duration: Each marker position is held for N frames before jumping to a new pseudo-random location.
  • Placement modes: Restrict marker positions to corners, edges, center zone, or a custom margin band.
  • Compression resistant: High-contrast markers survive lossy codecs at the chosen opacity.
